# ExamMeAi
## Description
ASP.NET Core MVC application for open question examps.
Stores subjects, domains, answers and AI assessment and grade on Microsoft Server db.

## Tech stuck
1. ASP.Net Core
2. Entity Framework Core (ORM)
3. SQL Server
4. NLog
5. Cookie Authentication
6. OpenAI
7. AJAX
8. vanilla js style
9. Razor
10. MVC, DI, Singleton
  
## Completed development issues
1. Scaffolding view and controllers for custom Database model (ORM)
2. EFCore - db data seeds with right syntax.
3. Initializing, updating and debuging db EF Core migration.
4. GUI lists, forms, buttons and layout customisation for design principles
5. Requestst with AJAX and vanilla JS for inner ASP MVC solution actions.
6. Service for user Authentication with Cookie
7. ViewComponent for view controll depending on login status
8. Task for OpenAI language model external service
